# URL Shortener with Authentication & Authorization



A full-stack **URL shortening application** built with **Node.js, Express, MongoDB, HTML, CSS, and JavaScript**, featuring secure login/signup and personalized URL management. Users can generate short URLs, copy them, open them, and track click countsâ€”all with their own dashboard.

---

## **Features**
- **User Authentication:** Secure login and signup using JWT.
- **Authorization:** Each user can view only their own URLs.
- **URL Shortening:** Generate short URLs from long URLs.
- **Copy & Open:** Easily copy short URLs or open them directly.
- **Click Tracking:** Track the number of times each short URL is accessed.
- **Responsive Design:** Works on both desktop and mobile devices.

---

## **Tech Stack**
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** JWT (JSON Web Tokens)
